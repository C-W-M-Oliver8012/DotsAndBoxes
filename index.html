About the game

Dots and Boxes is a game where players connect lines in order to form boxes. The player to successfully complete a box "wins" that box and score a point. Once a point has been scored, that player is allowed to go again.

How to AI works

The AI uses the minimax algorithm to search the game tree in order to find the best move. Alpha beta pruning is used to to disregard game trees that will not yield good results, thus speeding up the game tree search.

Future improvements

Everything seems to work exactly how I would expect it to, however, there is no doubt in my mind that there are bugs in my code. Much work needs to be done with the data structures in order to create a scalable version of this game. The code is is currently undocumented and the result of a day and a half long code sprint.
